## 🚀 はじめに | Introduction

**DHF Agent** は、革新的なブラウザRPA（ロボティック・プロセス・オートメーション）ツールです。最新のAI技術を活用してWebページを理解し、ユーザーが自動化ワークフローを構築するのを支援します。

他のAIエージェントとは異なり、DHF Agentは **「AI生成、ローカル実行」** モデルを採用しています。ワークフローが一度構築されると、効率的な固定コード実行に変換され、**トークンを消費することはありません**。これにより、実行の正確性が保証されるだけでなく、ゼロコストでの運用が実現します。反復作業から解放されたいオフィスワーカーであれ、効率的なWebインターフェースを求める開発者であれ、DHF Agentはあなたの最高のパートナーとなります。

## 🎯 なぜ DHF Agent を選ぶのか？ | Who is this for?

私たちはさまざまなユーザー向けにソリューションを最適化しました：

### 1. 👩‍💼 オフィスワーカー & 管理スタッフ (Office Workers)
**繰り返しのクリックに別れを告げ、時間を創造的な作業に使いましょう。**
*   **課題：** 毎日ブラウザでフォームに入力し、データをスクレイピングし、ボタンをクリックする作業の繰り返し。
*   **DHFの解決策：** やりたいことをAIに伝えるだけで、操作プロセスを自動生成します。内蔵の **タスクスケジューラー** を使えば、深夜にレポートのダウンロードやデータ入力を自動的に完了させ、朝出社したときには結果を確認するだけです。

### 2. 👨‍💻 AIギーク & 開発者 (Developers & Geeks)
**ゼロコストで高精度のWeb操作ソリューション。**
*   **課題：** 純粋なLLM駆動のWeb操作はトークンを大量に消費し、ハルシネーション（幻覚）が発生しやすく、不安定です。
*   **DHFの解決策：** AIを活用してPlaywrightスクリプトの作成を支援します。デバッグが通過すれば、その後の実行における **トークン消費は0** です。不確定なAI推論を確定的なコード実行に変換し、正確で信頼性が高く、無料です。

### 3. 🦄 一人起業家 & 個人開発者 (Solopreneurs)
**一人でもチームのような生産性を。**
*   **課題：** リソース不足、従業員を雇うコストが高い。
*   **DHFの解決策：** 異なるDHF Agentの役割（例：カスタマーサポートアシスタント、SNS運用担当、財務担当）を作成します。彼らはあなたのデジタル社員のように、それぞれのWebタスクをバックグラウンドで独立して完了し、あなたのビジネスを24時間365日稼働させます。

### 4. 🔗 MCP & OpenClaw ユーザー (Ecosystem Users)
**すべてをつなぐスーパーテンタクル。**
*   **課題：** 大規模言語モデルではWebページを正確に操作できない、または操作コストが高すぎる。
*   **DHFの解決策：** **MCP (Model Context Protocol)** を完全にサポートしています。DHF AgentはOpenClawやその他のAIツールの「手」として機能し、自然言語の指示を正確なPlaywrightのアクションに変換します。これにより、トークンを大幅に節約し、操作の成功率を向上させます。

## ✨ 主な機能 | Key Features

### ⚡ 10倍の効率向上
面倒で繰り返しの多いWeb操作をDHF Agentに任せることで、効率を10倍に飛躍させます。機械の仕事は機械に、人の仕事は人に。

### 🧠 AI駆動 & デバッグ支援
面倒なコードに別れを告げましょう。AIがワークフローの生成とデバッグをインテリジェントに支援し、自然言語でプロセス構築を可能にします。自動化開発はかつてないほど簡単になり、コードがわからなくても始められます。

### 💸 ランニングコストゼロ (Zero Runtime Cost)
これが私たちの最大の強みです。デバッグ完了後、ワークフローはAIの介入なしに独立して実行できます。**継続的なAPIトークンコストは発生しません**。真の経済性を実現します。

### 🔒 プライバシー優先
完全にローカルで実行され、データは暗号化して保存されます。ブラウザデータやアカウントパスワードはすべて自分のデバイス上にあり、完全に管理できるため、絶対的な安全性が確保されます。

### 🌐 MCPエコシステム連携
標準MCPプロトコルをサポートし、**n8n**、**Dify**、**OpenClaw** などの外部エコシステムとシームレスに接続します。アプリケーションの孤立を打破し、スーパーオートメーションワークフローを構築します。

### 🤝 ワークフロー共有
タスクフローのワンクリックエクスポートと共有をサポートします。あなたの「自動化の経験」をファイルに変換して同僚やコミュニティと共有し、能力の迅速な再利用を実現します。

### ⏱️ インテリジェントなタスクスケジューリング
秒単位の設定をサポートするプロフェッショナルなCronスケジューリングエンジンを内蔵しています。定期的な作業（例：毎日午前9時に自動打刻、毎週金曜日に週報を自動送信）を完全に自動化します。

---

## 🛠️ 技術統合 (For Developers)

DHF Agentは単なるツールではなく、インフラストラクチャです。

*   **MCPサーバーとして:** DHF AgentをCursor、Claude Desktop、またはその他のAIコードツールのMCPサーバーとしてマウントできます。これにより、AIプログラミングアシスタントが実際のブラウザを直接操作する能力を持つようになります。
*   **Playwrightコンバーター:** 不安定な自然言語の意図を、安定したPlaywrightスクリプトに定着させることができます。
*   **APIトリガー:** ローカルAPI呼び出しをサポートし、既存のPython/Node.jsプロジェクトに簡単に統合できます。

## 🚀 クイックスタート | Quick Start

1.  **ダウンロードとインストール**：DHF Agentをインストールします。
2.  **タスク作成**：「新規タスク」をクリックし、ターゲットURLを入力します。
3.  **AIレコーディング**：ダイアログボックスでAIに「ログインして、最近の注文Excelをダウンロードして」と伝えると、AIが自動的にステップを生成します。
4.  **保存と実行**：ステップが正しいことを確認し、タスクを保存します。タイマーを設定すれば、後は自動で作業してくれます。
