## 🚀 소개 | Introduction

**DHF Agent**는 혁신적인 브라우저 RPA(로봇 프로세스 자동화) 도구입니다. 고급 AI 기술을 사용하여 웹 페이지를 이해하고 사용자가 자동화된 워크플로우를 구축하도록 돕습니다.

다른 AI 에이전트와 달리 DHF Agent는 **"AI 생성, 로컬 실행"** 모델을 채택합니다. 워크플로우가 구축되면 효율적인 고정 코드 실행으로 변환되어 **토큰을 더 이상 소비하지 않으며**, 실행의 정확성을 보장하고 제로 비용 운영을 실현합니다. 반복적인 노동에서 벗어나고 싶은 직장인이든 효율적인 웹 인터페이스를 찾는 개발자이든 DHF Agent는 최고의 파트너입니다.

## 🎯 왜 DHF Agent를 선택해야 하나요? | Who is this for?

다양한 그룹을 위해 맞춤형 솔루션을 제공합니다:

### 1. 👩‍💼 사무직 및 관리 직원 (Office Workers)
**반복적인 클릭과 작별하고 창의적인 일에 시간을 쓰세요.**
*   **문제점:** 매일 브라우저에서 양식을 작성하고, 데이터를 스크랩하고, 버튼을 클릭하는 반복 작업.
*   **DHF 솔루션:** AI에게 원하는 것을 말하면 작업 프로세스가 자동으로 생성됩니다. 내장된 **작업 스케줄러**를 사용하면 심야에 컴퓨터가 자동으로 보고서 다운로드나 데이터 입력을 완료하도록 하고, 아침에 출근하여 결과를 바로 확인할 수 있습니다.

### 2. 👨‍💻 AI 괴짜 및 개발자 (Developers & Geeks)
**비용 제로, 고정밀 웹 작업 솔루션.**
*   **문제점:** 순수 LLM 기반 웹 작업은 토큰을 많이 소모하며 환각(Hallucination)이 발생하기 쉽고 불안정합니다.
*   **DHF 솔루션:** AI를 사용하여 Playwright 스크립트 작성을 지원합니다. 디버깅이 통과되면 후속 실행은 **토큰 소비가 0**입니다. 불확실한 AI 추론을 확정적인 코드 실행으로 변환하여 정확하고 신뢰할 수 있으며 무료입니다.

### 3. 🦄 1인 기업 및 독립 개발자 (Solopreneurs)
**한 사람이 하나의 팀입니다.**
*   **문제점:** 자원 부족, 직원 고용 비용 부담.
*   **DHF 솔루션:** 다양한 DHF Agent 역할(예: 고객 서비스 지원, 소셜 미디어 운영, 재무 전문가)을 만듭니다. 그들은 디지털 직원처럼 백그라운드에서 각자의 웹 작업을 독립적으로 완료하여 비즈니스가 24/7 운영되도록 합니다.

### 4. 🔗 MCP 및 OpenClaw 사용자 (Ecosystem Users)
**모든 것을 연결하는 슈퍼 촉수.**
*   **문제점:** 대규모 모델은 웹 페이지를 정밀하게 조작할 수 없거나 운영 비용이 너무 비쌉니다.
*   **DHF 솔루션:** **MCP(Model Context Protocol)**를 완벽하게 지원합니다. DHF Agent는 OpenClaw 또는 다른 AI 도구의 "손"으로 마운트되어 자연어 명령을 정밀한 Playwright 동작으로 변환하여 토큰을 크게 절약하고 작업 성공률을 높일 수 있습니다.

## ✨ 주요 기능 | Key Features

### ⚡ 10배 효율성 향상
번거롭고 반복적인 웹 작업을 DHF Agent에 맡겨 효율성을 10배 높이세요. 기계의 일은 기계에게, 사람의 일은 사람에게 맡기세요.

### 🧠 AI 기반 및 보조 디버깅
지루한 코드와 작별하세요. AI가 지능적으로 워크플로우 생성 및 디버깅을 지원하여 자연어로 프로세스를 구축할 수 있습니다. 자동화 개발이 그 어느 때보다 쉬워졌으며 코드를 몰라도 시작할 수 있습니다.

### 💸 제로 런타임 비용 (Zero Runtime Cost)
이것이 우리의 가장 큰 장점입니다. 디버깅이 완료되면 워크플로우는 AI 개입 없이 독립적으로 실행될 수 있습니다. **지속적인 API 토큰 비용 없음**, 진정한 경제성을 실현합니다.

### 🔒 프라이버시 우선
완전히 로컬에서 실행되며 데이터는 암호화되어 저장됩니다. 브라우저 데이터, 계정 비밀번호는 모두 자신의 기기에 있으며 완전한 제어권을 가지고 있어 절대적인 안전을 보장합니다.

### 🌐 MCP 생태계 상호 연동
표준 MCP 프로토콜을 지원하여 **n8n**, **Dify**, **OpenClaw** 등 외부 생태계와 원활하게 연결됩니다. 애플리케이션 고립을 타파하고 슈퍼 자동화 워크플로우를 구축합니다.

### 🤝 워크플로우 공유
작업 흐름의 원클릭 내보내기 및 공유를 지원합니다. "자동화 경험"을 파일로 변환하여 동료나 커뮤니티와 공유하여 기능을 빠르게 재사용할 수 있습니다.

### ⏱️ 지능형 작업 스케줄링
초 단위 타이밍 구성을 지원하는 전문가급 Cron 스케줄링 엔진 내장. 주기적인 작업(예: 매일 오전 9시 자동 출근 체크, 매주 금요일 주간 보고서 자동 발송)을 완벽하게 자동화합니다.

---

## 🛠️ 기술 통합 (For Developers)

DHF Agent는 단순한 도구가 아니라 인프라입니다.

*   **MCP 서버로서:** DHF Agent를 Cursor, Claude Desktop 또는 기타 AI Code 도구의 MCP 서버로 마운트할 수 있습니다. 이를 통해 AI 프로그래밍 도우미가 실제 브라우저를 직접 조작할 수 있는 기능을 갖게 됩니다.
*   **Playwright 변환기:** 불안정한 자연어 의도를 안정적인 Playwright 스크립트로 침전시킬 수 있습니다.
*   **API 트리거:** 로컬 API 호출을 지원하여 기존 Python/Node.js 프로젝트에 쉽게 통합할 수 있습니다.

## 🚀 빠른 시작 | Quick Start

1.  **다운로드 및 설치**: DHF Agent를 설치합니다.
2.  **작업 생성**: "새 작업"을 클릭하고 대상 URL을 입력합니다.
3.  **AI 기록**: 대화 상자에서 AI에게 "로그인하고 최근 주문 Excel을 다운로드해 줘"라고 말하면 AI가 자동으로 단계를 생성합니다.
4.  **저장 및 실행**: 단계가 올바른지 확인하고 작업을 저장합니다. 타이머를 설정하면 앞으로 자동으로 작업이 수행됩니다.
